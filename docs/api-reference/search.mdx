# Search

The `/search` endpoint lets you search documents indexed by Scrapix through your account's default Meilisearch engine. It resolves the correct index automatically from the target URL.

<Note>
  Requires a [default Meilisearch engine](/api-reference/engines#post-enginesiddefault) to be configured for your account.
</Note>

## POST /search

Search documents for a given domain.

**Cost:** 2 credits per request

### Request Body

```json
{
  "url": "https://docs.example.com",
  "q": "getting started",
  "limit": 10,
  "offset": 0,
  "filter": "lang = 'en'",
  "sort": ["_rankingScore:desc"]
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `url` | string | Yes | Target domain URL — used to resolve the Meilisearch index UID |
| `q` | string | Yes | Search query (non-empty) |
| `limit` | integer | No | Maximum number of results to return |
| `offset` | integer | No | Number of results to skip (for pagination) |
| `filter` | string \| object | No | Meilisearch [filter expression](https://www.meilisearch.com/docs/reference/api/search#filter) |
| `sort` | string[] | No | Meilisearch [sort rules](https://www.meilisearch.com/docs/reference/api/search#sort) |

### Example

```bash
curl -X POST https://scrapix.meilisearch.dev/search \
  -H "x-api-key: sk_live_abc123..." \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://docs.example.com",
    "q": "authentication",
    "limit": 5
  }'
```

### Response (200 OK)

Returns the raw Meilisearch search response:

```json
{
  "hits": [
    {
      "id": "doc_abc123",
      "title": "Authentication Guide",
      "url": "https://docs.example.com/auth",
      "content": "Learn how to authenticate with..."
    }
  ],
  "query": "authentication",
  "processingTimeMs": 12,
  "limit": 5,
  "offset": 0,
  "estimatedTotalHits": 23
}
```

### How Index Resolution Works

The `url` field is used to derive a Meilisearch index UID. This is the same index UID that was used when the domain was originally crawled. You don't need to know the exact index name — just provide the same domain URL.

For example, if you crawled `https://docs.example.com` with Scrapix, searching with `"url": "https://docs.example.com"` will automatically route to the correct index.

### Error Responses

| Status | Code | Description |
|--------|------|-------------|
| 400 | `bad_request` | Invalid URL scheme (must be http/https) or empty query |
| 401 | `unauthorized` | Missing or invalid credentials |
| 402 | `insufficient_credits` | Account has fewer than 2 credits |
| 404 | `not_found` | No default Meilisearch engine configured for account |
